3.4 Testing Digital Software

The testing of digital software is entirely dependant on the complexity involved in the software. If the software has lots of if..else conditions embedded within it then there will need to be a plan drawn up to test each combination of variables so all permutations are tested. When it comes to websites on the internet then plans need to be made to ensure the site can handle multiple different browsers and devices.
